# Anthropic Academy — Learning in Public

I'm working through every official Anthropic Academy course and building a real project for each track, rather than just watching and moving on. This repo is the map and the proof of work: the progress tracker, my notes, and links to the projects each track produces.

The goal is simple. Become genuinely expert on how Claude is built on and deployed, the way someone who works at Anthropic would be, and have the artifacts to show for it.

## The path

Eighteen courses across six phases, each mapped to something I build. Live status lives in [PROGRESS.md](PROGRESS.md).

1. **Foundations** — Claude 101, AI Fluency, AI Capabilities and Limitations
2. **Developer core** — Building with the Claude API, Claude Code 101, Claude Code in Action
3. **Agentic and extensibility** — MCP (intro and advanced), agent skills, subagents
4. **Deployment** — Claude on Amazon Bedrock and Google Vertex AI
5. **Product surface** — Claude Cowork
6. **AI Fluency** — the teaching track

Plus the official material beyond the Academy: the [anthropics/courses](https://github.com/anthropics/courses) repo, the cookbook, and the engineering and research blogs.

## What each track ships

Every track ends in a real artifact, not a checkbox. Some become their own repos, some land here as notes and code. The point is to be able to explain and rebuild anything I claim to know.
